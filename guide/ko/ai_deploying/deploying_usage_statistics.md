웹 포인트 추론 요청을 하게 되면 내부 통계 모듈을 통해 추론 결과 데이터를 수집합니다.

웹 포인트에서 추론 후 수집된 통계 데이터는 웹 포인트 → 사용자 추론 서버 포인트 → 통계 탭에서 확인할 수 있습니다.
